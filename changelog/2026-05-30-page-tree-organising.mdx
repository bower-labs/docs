The sidebar page tree is now fully organisable. You can put your workspace in exactly the order you want and act on any item without opening it.

## Drag to reorder — Notion-style

Grab any note or collection and drag it. A blue line shows where it will land:

- **Drop between two items** to reorder it within that level.
- **Drop onto a collection** to move it inside. Collapsed collections **auto-expand** after a moment when you hover over them, so you can drag deep into the tree in one motion.

Reordering works at every level of the hierarchy, for both notes and collections. The order is **shared across the workspace** — everyone on your team sees the same arrangement.

## A ⋮ menu on every item

Hover over any item in the page tree and a **⋮** menu appears. It gives you per-item actions without opening the item first:

| Action | What it does |
| --- | --- |
| New collection | Create a collection nested inside this one *(collections only)* |
| Rename | Edit the title inline |
| Move | Move the item to another collection |
| Copy | Duplicate the item to a collection |
| Copy link | Copy a direct link to the item |
| Manage access | Open access controls for the item |
| Delete | Move to trash |

## A tidier sidebar

The sidebar now groups things more clearly: a collapsible **Library** (Collections, Artifacts, Protocols, and Equipment), your **Collections** page tree, and a **Recents** list of what you've touched lately. Trash stays pinned at the bottom.
